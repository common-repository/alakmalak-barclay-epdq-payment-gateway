The AM Barclay ePDQ Payment plugin enables merchants that use Woocommerce to process online card payments using Barclay Gateway.

== Description ==

Transform your website's payment capabilities with the AM Barclay ePDQ Payment Gateway plugin. This integration offers seamless transactions, optimizing the Barclay ePDQ experience for secure and efficient payments.

Enhance your website's payment efficiency with the AM Barclay ePDQ Payment Gateway Plugin — a seamless integration that ensures secure transactions and optimizes the Barclay ePDQ experience.

== Key Features: ==

* Secure Transactions: Safely process payments using the Barclay ePDQ platform.
* Effortless Integration: Integrate seamlessly with your website for a streamlined checkout process.
* Enhanced Efficiency: Improve payment processing efficiency to enhance customer satisfaction.
* With AM Barclay ePDQ Payment Gateway, you can ensure secure and efficient online transactions, enhancing your website's payment functionalities.
* SHA-IN/OUT support
* Configure Return on Redirection
* Configurable for test and production accounts simultaneously for easy mode switching.
* SHA-IN Passphrase : Option to set Sha in pass phrase as per your barclay merchant panel.
* SHA-OUT Passphrase : Option to set Sha Out pass phrase as per your barclay merchant panel.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the 'alakmalak-barclay-epdq-payment-gateway' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Admin Dashboard in wordpress
4. Activate the plugin through the ‘Plugins’ menu in WordPress
5. Go to WooCommerce > Settings > Payment Gateways > AM Barclay Gateway and set the parameters.
